수정해야 하는 항목입니다.

<Token.txt> : 봇의 TOKEN을 입력해 주세요.
<APPLICATION_ID.txt> : 봇의 APPLICATION ID를 입력해 주세요.

[ 봇 생성 방법 ]
1. https://discord.com/developers/applications 에 접속합니다.
2. 오른쪽 상단에 NEW Application을 클릭하고 봇 이름 작성 및 약관 동의를 클릭하고 생성합니다.
3. 생성한 봇을 클릭하고 왼쪽 SETTINGS → Bot 카테고리에서 Reset Token을 클릭하고 생성합니다. 생성한 토큰을 복사하고 Token.txt에 붙여 넣습니다.
4. Authorization Flow 의 PUBLIC BOT, Privileged Gateway Intents의 PRESENCE INTENT, SERVER MEMBERS INTENT, MESSAGE CONTENT INTENT를 켭니다.
5. SETTINGS → General Information 카테고리에서 APPLICATION ID를 APPLICATION_ID.txt에 붙여 넣습니다.
6. 아래 주소는 디스코드 서버에 봇을 추가하는 주소입니다. client_id= 다음부분에 APPLICATION ID를 입력하고 인터넷 주소창에 입력해 봇을 추가합니다.

https://discord.com/oauth2/authorize?client_id=&permissions=8&scope=bot 
예시) https://discord.com/oauth2/authorize?client_id=1234567891234567890&permissions=8&scope=bot 



[ 설치 하셔야 하는 파일 ]

python-3.11.6-amd64 파일을 실행해 파이썬 설치를 진행합니다.
설치 과정에서 반드시 PIP 설치가 진행되어야 합니다.

모든 설치가 다 완료되면 start.bat파일을 실행해 봇을 실행합니다.


[ 기능 ]

* 관리자는 경매 아이템을 등록, 제거 할 수 있습니다.

* 등록된 아이템은 해당 일자의 자정에 경매를 진행합니다.
* 등록된 아이템은 봇을 재부팅 하더라도 기록이 됩니다.

* 경매는 /입찰 명령어를 통해 진행할 수 있으며, 경매 마감 1시간 전 모든 유저에게 알림을 보냅니다.
* 경매는 다음날 자정에 마감되며, 마감 시 경매 당첨자를 알려드립니다.



[ 명령어 ]

1. 입찰 <경매상품> <경매가> : 경매 아이템 입찰을 진행합니다.
	- 경매상품 : 경매 아이템 이름을 입력합니다. (자동완성)
	- 경매가 : 경매가를 입력합니다.

	* 경매일 당일 입찰을 진행합니다.
	* 반드시 마지막 경매가보다 100 다이아 높은 가격을 불러야합니다.


2. 경매 채널 <채널> : [관리자] 경매 채널을 지정합니다.
	- 채널 : 경매 채널을 지정합니다. (자동완성)

	* 경매 전용 채널을 지정합니다.
	* 봇 사용을 위해 반드시 지정하셔야 합니다.
	* 최초 1회 사용으로 봇을 재부팅 하더라도 영구적으로 사용 가능합니다.


3. 경매 등록 <상품이름> <날짜> : [관리자] 새로운 경매 일정을 등록합니다.
	- 상품이름 : 상품이름을 입력합니다.
	- 날짜 : 날짜를 선택합니다. (자동완성)

	* 경매 아이템을 등록합니다.
	* 한 일자에 대해 중복된 아이템을 등록할 수 없습니다.


4. 경매 제거 <상품이름> : [관리자] 등록된 경매 일정을 제거합니다.
	- 상품이름 : 상품이름을 입력합니다. (자동완성)

	* 경매 상품을 제거합니다.
	* 자동완성은 "<아이템이름> (일자)"로 표시됩니다.


5. 경매 현황 <날짜> : [관리자] 등록된 경매 일정을 확인합니다.
	- 날짜 : 날짜를 선택합니다. (자동완성)

	* 입력한 날짜에 등록된 아이템을 확인합니다.


- 버그가 있을 수 있습니다, 시스템상 버그는 무료로 수정해 드립니다.
